# SECURITY.md — 安全规则

## API Key 管理

- 所有密钥通过环境变量注入，不允许硬编码
- `.env` 文件在 `.gitignore` 中，永远不提交
- 生产环境使用 secrets manager（AWS Secrets Manager / HashiCorp Vault）

## 输入安全

- Prompt Injection 检测（正则模式匹配）
- 输入长度限制（默认 10000 字符）
- 速率限制（默认 60 RPM）

## 输出安全

- PII 自动检测和脱敏（邮箱、电话、SSN、信用卡号）
- 内容审核（有害内容检测）
- 系统提示词泄露防护

## 工具安全

- 每个工具独立权限标注
- 代码执行在沙箱中（Docker 容器）
- 工具超时控制（默认 30s）
- 工具结果长度限制（默认 10000 字符）

## 审计

- 所有工具调用记录审计日志
- 所有 LLM 调用记录 token 用量和延迟
- 安全事件（injection 检测、PII 泄露）结构化日志

## 数据

- 用户对话数据本地存储，不上传第三方
- 向量数据库数据加密存储
- 定期清理过期会话数据
