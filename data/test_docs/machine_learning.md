# 机器学习基础

机器学习（Machine Learning）是人工智能的一个分支，它使计算机系统能够通过经验自动改进。

## 主要类型

### 监督学习
- 使用标注数据训练
- 常见算法：线性回归、决策树、随机森林、SVM、神经网络
- 应用：分类、回归

### 无监督学习
- 使用未标注数据
- 常见算法：K-Means、DBSCAN、PCA、自编码器
- 应用：聚类、降维、异常检测

### 强化学习
- 通过与环境交互学习
- 常见算法：Q-Learning、PPO、A3C
- 应用：游戏AI、机器人控制

## 深度学习

深度学习是机器学习的子集，使用多层神经网络：

| 模型类型 | 应用场景 | 代表模型 |
|---------|---------|---------|
| CNN | 图像识别 | ResNet、VGG |
| RNN/LSTM | 序列数据 | Transformer |
| GAN | 生成模型 | StyleGAN |
| Transformer | NLP/CV | BERT、GPT |

## 评估指标

- **分类**：准确率、精确率、召回率、F1-score
- **回归**：MSE、RMSE、MAE、R²
- **聚类**：轮廓系数、Calinski-Harabasz指数
